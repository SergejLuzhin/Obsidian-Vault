## Задача 1 - Фильтрация + сбор в список

**Условие:**
Есть список чисел:
```java
List<Integer> nums = List.of(1, 5, 10, 2, 15);
```
Оставить только числа больше 5.

**Решение:**
```java
List<Integer> nums = List.of(1, 5, 10, 2, 15);  
  
List<Integer> newNums = nums.stream()  
        .filter(num -> num > 5)  
        .collect(Collectors.toList());  
  
System.out.println(newNums);
//[10, 15]
```

## Задача 2 — map() + toUpperCase

**Условие:**
Есть список строк:
```java
List<String> names = List.of("anna", "bob", "kate");
```
Перевести все строки в верхний регистр.

**Решение:**
```java
List<String> names = List.of("anna", "bob", "kate");  
  
List<String> capitalNames = names.stream()  
        .map(String::toUpperCase)  
        .toList();  
  
System.out.println(capitalNames);
//["ANNA", "BOB", "KATE"]
```

## Задача 4 — сортировка с условием

**Условие:**
Есть список имён:
```java
List<String> names = List.of("alex", "tom", "benjamin", "bob");
```
Оставить строки длиной > 3, перевести в upper case, отсортировать.

**Решение:**
```java
List<String> names = List.of("alex", "tom", "benjamin", "bob");  
  
List<String> newNames = names.stream()  
        .filter(name -> name.length() > 3)  
        .map(String::toUpperCase)  
        .sorted()  
        .toList();  
  
System.out.println(newNames);
// ["ALEX", "BENJAMIN"]
```

## Задача 5* — группировка по значению (частотность)

**Условие:**
Есть список слов:
```java
List<String> words = List.of("a", "b", "a", "c", "b", "a");
```
Посчитать количество появлений каждого слова и вернуть map.

**Решение:**
```java
List<String> words = List.of("a", "b", "a", "c", "b", "a");  
  
Map<String, Long> frequencyMap = words.stream()  
        .collect(Collectors.groupingBy(  
                word -> word,  
                Collectors.counting()  
        ));  
  
System.out.println(frequencyMap);
//{a=3, b=2, c=1}
```

## Задача 6 — список списков → один список (flatMap)

**Условие:**
Есть список списков:
```java
List<List<Integer>> data = List.of(
    List.of(1,2),
    List.of(3,4),
    List.of(5)
);
```
Сделать один список всех чисел.

**Решение:**
```java
List<List<Integer>> data = List.of(  
        List.of(1,2),  
        List.of(3,4),  
        List.of(5)  
);  
  
List<Integer> flatData = data.stream()  
        .flatMap(list -> list.stream()) //Collection::stream
        .toList();  
  
System.out.println(flatData);
//[1, 2, 3, 4, 5]
```

## Задача 7 — второй по величине элемент

**Условие:**
Есть список:
```java
List<Integer> nums = List.of(5, 1, 7, 7, 3, 9);
```
Найти второй максимальный уникальный элемент.

**Решение:**
```java
List<Integer> nums = List.of(5, 1, 7, 7, 3, 9);  
  
Optional<Integer> secondMax = nums.stream()  
        .distinct()  
        .sorted(Comparator.reverseOrder())  
        .skip(1)  
        .findFirst();  
  
secondMax.ifPresent(System.out::println);
//7
```

## Задача 8 — преобразование списка объектов в Map

**Условие:**
Есть класс и список:
```java
class User {
    int id;
    String name;
    User(int id, String name) { this.id = id; this.name = name; }
}

List<User> users = List.of(
    new User(1, "Max"),
    new User(2, "Anna"),
    new User(3, "Kate")
);
```
Преобразовать в Map<id, name>.

**Решение:**
```java
Map<Integer, String> usersMap = users.stream()  
        .collect(Collectors.toMap(  
                user -> user.id,  
                user -> user.name  
        ));  
  
System.out.println(usersMap);
//{1=Max, 2=Anna, 3=Kate}
```

## Задача 9* — фильтрация + сортировка объектов + map()

**Условие:**
Есть класс и список:
```java
class Product {
    String name;
    int price;
    Product(String name, int price) { this.name = name; this.price = price; }

List<Product> products = List.of(
    new Product("Phone", 900),
    new Product("TV", 1200),
    new Product("Mouse", 10),
    new Product("Laptop", 1500)
);
```

Выбрать товары с ценой > 100,
отсортировать по цене по возрастанию,
вернуть список их имён.

**Решение:**
```java
List<String> expensiveProducts = products.stream()  
        .filter(product -> product.price > 100)  
        .sorted(Comparator.comparingInt(product -> product.price))  
        .map(product -> product.name)  
        .toList();  
  
System.out.println(expensiveProducts);
//["Phone", "TV", "Laptop"]   // 900, 1200, 1500
```

## Задача 10 — из предложений → список слов (flatMap)

**Дано:**
```java
List<String> sentences = List.of(
    "java is cool",
    "streams are powerful",
    "practice makes perfect"
);
```
Разбить каждую строку на слова и собрать в один список.

**Решение:**
```java
List<String> words = sentences.stream()  
        .flatMap(sentence -> Arrays.stream(sentence.split(" ")))  
        .toList();  
  
System.out.println(words);
//["java", "is", "cool",
// "streams", "are", "powerful",
// "practice", "makes", "perfect"]
```
