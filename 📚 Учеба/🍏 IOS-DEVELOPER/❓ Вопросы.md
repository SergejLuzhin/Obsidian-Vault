## В чем разница между shadowing'ом переменной и optional binding?

Чем отличаются [[07.  Опционалы#Распаковка опицонала (unwrapping)|механизмы распаковки опционалов]] shadowing переменной и optional binding с краткой записью (без создания новой переменной), ведь по сути это одно и то же. Мы можем использовать распакованную переменную только внутри блока if.

`Optional binding:`
```swift
var message: String?
if let message {
    print(message)
}
```

`Shadowing переменной`
```swift
if let message = message {
    ...
}

или

guard let message = message else {
    return
}
...
```
